\section{Install}
\label{sec:orgb84023d}

Set the \texttt{nix-folder} in \href{./flake.nix}{flake.nix} (the result of \texttt{pwd})

Configure your hosts in \href{./hosts}{hosts}.

Rebuild your system : \texttt{sudo nixos-rebuild switch -{}-{}flake /path/to/this/folder}
