#+author: Gabriel TESSIER

** Modules

I have my modules separated into two subdirectories:
- System-level - stored in the [[./modules/system][system directory]]
- User-level - stored in the [[./modules/user][user directory]]

** Hosts

Configurations for all machines are in [[./hosts][hosts]].

** Install

Run =sudo nix run --option experimental-features "nix-command flakes" nixpkgs#nixos-facter -- -o facter.json=

Set the =nix-folder= in [[./flake.nix][flake.nix]] (the result of =pwd=)

Configure your hosts in [[./hosts][hosts]].

Rebuild your system : =sudo nixos-rebuild switch --flake /path/to/this/folder=
